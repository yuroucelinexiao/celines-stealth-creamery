## Macros

Prioritizing pints that have high nutrient-to-calorie ratios (NCR) for satiety and macros.

<div align="center" style="margin: 3em auto;">
$$
\text{NCR Score} = \frac{\text{Protein (g)} + \text{Fiber (g)}}{\text{Total Calories}} \times 100
$$
</div>

<div class="no-border-table" align="center" style="margin: 0 auto;">

| **Stars** | **NCR Score** |
| :---: | :---: |
| <span style="font-size: 1.5em;">★★★★★</span> | >9 |
| <span style="font-size: 1.5em;">★★★★☆</span> | 7–8.9 |
| <span style="font-size: 1.5em;">★★★☆☆</span> | 5–6.9 |
| <span style="font-size: 1.5em;">★★☆☆☆</span> | 3–4.9 |
| <span style="font-size: 1.5em;">★☆☆☆☆</span> | 1–2.9 |
| <span style="font-size: 1.5em;">☆☆☆☆☆</span> | <1 |

</div>

Deductions:
- minus <span class="half-star">☆</span> for > 10g saturated fat
- minus <span class="half-star">☆</span> for > 20g added sugars
